# Serveeny - Services à domicile en Tunisie

Serveeny est une plateforme moderne permettant de trouver facilement les meilleurs prestataires de services en Tunisie (électriciens, plombiers, femmes de ménage, jardiniers, etc.).

## Fonctionnalités

- **Recherche de prestataires** : Trouvez des experts qualifiés près de chez vous.
- **Filtrage par catégorie** : Naviguez facilement parmi les différentes catégories de services.
- **Profils détaillés** : Consultez les avis, les tarifs et les informations de contact.
- **Favoris** : Sauvegardez vos prestataires préférés.
- **Multilingue** : Interface disponible en Français et Arabe.

## Technologies Utilisées

- **Framework** : React + Vite
- **Langage** : TypeScript
- **Styles** : Tailwind CSS
- **UI Components** : Shadcn UI
- **Icônes** : Lucide React
- **Carte/Localisation** : Intégration simple (MapPin)

## Installation et Démarrage

1. **Cloner le repository**
   ```bash
   git clone <URL_DU_REPO>
   cd Services
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

## Structure du Projet

- `src/components` : Composants réutilisables (UI, Layout, Home, Providers).
- `src/pages` : Pages principales de l'application.
- `src/contexts` : Gestion de l'état global (Langue, Thème, Favoris).
- `src/lib` : Utilitaires et constantes.
